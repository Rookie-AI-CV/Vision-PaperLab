## 笔记模板

1. **论文基本信息**
   - 作者、发表会议/期刊、年份
   - 论文链接

2. **核心贡献**
   - 主要创新点
   - 解决的问题

3. **方法概述**
   - 网络架构
   - 关键技术点

4. **实验结果**
   - 主要数据集上的性能
   - 关键实验分析

5. **个人思考**
   - 方法优缺点
   - 后续工作启发
